# Local fonts

- `Cygre-ExtraBold.ttf` — Cygre ExtraBold, используется в заголовках и логотипе. Источник: [Fontiko](https://fontiko.com/fonts/127/cygre-font/), автор RandomMaerks.
- `Inter-Cyrillic.woff2` и `Inter-Latin.woff2` — Inter 400–600, получены из [Google Fonts](https://fonts.google.com/specimen/Inter).

Шрифты лежат локально, поэтому preview и будущая WordPress-тема не зависят от внешнего CDN.
