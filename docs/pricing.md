# Секция тарифов

Статус: готова базовая адаптивная верстка по Figma.

## Сделано

- Три тарифа вынесены в переиспользуемый Pug-миксин `pricing-card`.
- Подключены фоновые SVG-ассеты и иконка звезды из Figma.
- Сохранены цвета карточек, панелей услуг, бейджей и кнопок.
- На desktop карточки идут в три колонки, на tablet — в две, на mobile — последовательно.
- Высота desktop-карточек плавно масштабируется, а mobile-карточки растягиваются по содержимому без лишнего пространства.
