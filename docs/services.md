# Секция дополнительных услуг

Статус: готова базовая адаптивная верстка по Figma.

## Сделано

- Секция собрана через переиспользуемый Pug-компонент `service-card`.
- На desktop сохранена исходная композиция: три карточки сверху, широкая карточка и две соседние снизу, отдельная правая колонка.
- Подключена локальная SVG-иконка звезды из Figma.
- Кнопки «Узнать детали» сделаны единым BEM-элементом с hover/focus-состояниями.
- На tablet/mobile карточки переходят в последовательную колонку без пустых grid-областей.
