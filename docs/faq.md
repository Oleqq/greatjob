# FAQ-секция

Статус: готова базовая адаптивная верстка и интерактивность.

## Сделано

- Две колонки: категории вопросов слева и отдельные панели аккордеонов справа.
- Три tab-группы: «Основное», «Процесс и условия», «Гарантии».
- Для каждой группы подготовлен собственный список вопросов и ответов.
- Использованы нативные `details/summary` и отдельный native JS-модуль для закрытия соседних вопросов.
- Добавлены SVG-иконки плюс/минус из Figma.
- Добавлена плавная CSS-анимация раскрытия/закрытия через `grid-template-rows: 0fr → 1fr`, без расчётов `height` в JS.
- Иконка состояния собрана из двух CSS-линий: плюс вращается, а вертикальная линия схлопывается в минус.
- При смене tab-группы активируется соответствующая панель и первый вопрос новой группы.
- На ширине до 900px tabs и аккордеон переходят в вертикальную последовательность.

## Осталось

- Утвердить финальные тексты вопросов и ответов.
- Провести финальную сверку на desktop/tablet/mobile ширинах.
- Перенести категории и вопросы в структуру полей WordPress/ACF.
