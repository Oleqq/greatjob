# Figma context

- Макет: [GreatJob](https://www.figma.com/design/GvYjV98aVqGmfqyVZEjx9a/GreatJob?node-id=1-1100&m=dev)
- Основной node: `1:1100`
- Desktop frame: `1600×8002`
- Контент: `1200 px`, боковое поле на desktop — `200 px`
- Desktop header: `52 px`
- Основные шрифты: `Cygre ExtraBold`, `Inter`

## Секции

1. Header и hero.
2. Решаем ваши проблемы.
3. План действий в цифрах.
4. Три шага до предложения о работе.
5. Отзывы о нашей работе.
6. Тарифы.
7. Список дополнительных услуг.
8. Вакансии в Telegram-канале.
9. Часто задаваемые вопросы.
10. Footer.

В макете нет отдельных mobile/tablet frames. Адаптивные состояния будут собираться от desktop-композиции без изменения смысловой структуры.
