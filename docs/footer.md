# Footer

Статус: готова базовая адаптивная верстка по Figma.

## Сделано

- Полноширинный тёмный footer с контентом внутри контейнера 1200px.
- Логотип подключён единым экспортированным ассетом `static/images/brand-logo.png`, без отдельного текстового элемента.
- Сохранены desktop-позиции логотипа, двух навигационных колонок, Telegram-текста и copyright.
- Добавлены hover-состояния ссылок и ссылка на Telegram-канал.
- На tablet/mobile блоки перестраиваются в вертикальную последовательность.
