# Секция отзывов

Статус: готова базовая статичная верстка и интерактивный слайдер.

## Сделано

- Добавлен Swiper.js через отдельный native JS-модуль.
- Карточка отзыва вынесена в переиспользуемый Pug-миксин `review-card`.
- Подключены локальные SVG-ассеты из Figma: кавычки, разделитель автора и рейтинги.
- Настроено отображение 1 / 2 / 4 карточек на mobile / tablet / desktop.
- Добавлены loop, drag/swipe, кнопки навигации, focus-visible и hover-состояния.
